#### 🔎 해시(Hash) 개념 요약
- **정의**: 데이터를 `키(key)`와 `값(value)` 쌍으로 저장하는 자료구조
- **특징**: `키 → 해시함수 → 인덱스`를 통해 값을 **빠르게 검색** 가능
- **시간 복잡도**:  
  - 평균: `검색`, `삽입`, `삭제` 모두 **O(1)**  
- **키를 해시 함수로 처리해 곧바로 저장 위치를 계산** → 리스트처럼 순차적으로 탐색할 필요 없이 **즉시 접근 가능**
